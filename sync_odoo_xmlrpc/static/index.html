Sync Odoo XML-RPC (Sale Order)

Este módulo permite sincronizar pedidos de venta (sale.order) desde una instancia local de Odoo hacia otra instancia remota mediante XML-RPC.

Características

  • Sincroniza pedidos de venta de Odoo a otra base de datos.
  • Valida si el pedido ya fue sincronizado para evitar duplicados.
  • Crea el cliente remoto si no existe (buscando por NIT/VAT o nombre).
  • Crea los productos remotos si no existen (usando el default_code).
  • Notificación amigable al usuario en Odoo con el resultado.

Configuración

En Ajustes → Parámetros del sistema, configurar las siguientes claves:

sync_odoo_xmlrpc.remote_odoo_url = https://mi-odoo-remoto.com
sync_odoo_xmlrpc.remote_odoo_db = mi_basededatos
sync_odoo_xmlrpc.remote_odoo_user = usuario@odoo.com
sync_odoo_xmlrpc.remote_odoo_password = mipassword

Uso

  1. Abrir un pedido de venta en Odoo.
  2. Hacer clic en el botón Sincronizar Pedido.
  3. El sistema validará y creará el pedido en la base remota.
