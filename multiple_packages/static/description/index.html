Multiple Packages

Pack one line into multiple packages

TO AVOID ANY ISSUES, PLEASE, USE ALWAYS LATEST VERSION FROM OUR GITHUB REPOSITORY - https://github.com/ventor-tech/merp/tree/17.0

This module extends packing flow in Inventory and allows you to split a single stock move line into multiple packages in a convenient wizard.

  • Create several packages from one move line in one action
  • Define number of packages and items per package
  • Select Package Type (filtered by the selected Delivery Carrier)
  • Optionally set Shipping Weight per package type

Designed for fast warehouse operations and compatible with delivery carriers configuration.

How it works

Open the wizard from a move line and specify:

  • Number of Packages to create
  • Items per Package
  • Package Type

After confirmation the module creates packages and splits the move line quantity accordingly.

Carrier-based Package Type filtering

When a delivery carrier is set on the picking, the wizard filters available Package Types so the user selects only relevant package types for that carrier.

This helps to avoid mistakes and keeps packing aligned with carrier requirements.
