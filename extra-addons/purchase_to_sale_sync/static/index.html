Purchase to Sale Sync

Este módulo permite sincronizar automáticamente órdenes de compra como órdenes de venta en otra instancia Odoo usando XML-RPC.

Características:

  • Conexión entre instancias Odoo vía XML-RPC
  • Configuración desde Ajustes Generales
  • Búsqueda de productos por código interno (`default_code`)
  • Creación automática del cliente remoto si no existe

Configuración:

Después de instalar el módulo, dirígete a Ajustes > Configuración General y completa los datos de conexión:

  • URL del Odoo remoto
  • Base de datos
  • Usuario
  • Contraseña

Ejemplo de configuración:

Flujo:

  1. Crear una orden de compra.
  2. El sistema se conecta al Odoo remoto.
  3. Busca cliente y productos por nombre/código.
  4. Crea una orden de venta correspondiente.
