Portal Sale Management

DeeB

Overview

odoo App will provide sales functionality. internally and externally from portal screen You can create quotation, Only customers appended for the sales person can be selected, portal user can create view and download sale order, ..

Feature

User create quotation from internal system.
Set portal user as salesman on customer.
Portal User create quotation from portal easy as in internal.
Portal user can view his quotation.
Portal user can view his sales order.
Portal user can create quotation from inside sales order form or tree view.
Portal user can create quotation from inside Quotation form or tree view.
Portal user can print quotation from inside quotation form or Sale order form
Portal user can download quotation from inside quotation form or Sale order form

How To Work

Sales Manger open customers and set salesman over customer
reset portal user password
Portal salesman open my account portal website menu
Select Quotation menu to show all quotations created by salesman
Create new quotation
Select customer "only your customers can be selected" and order date then add lines Note you can filter product by the name or internal code
insert all data and click save
When the record saved the page rendered in your quotations
Click on record to display details
User can print or download the sales quotation from portal
Internal sales manger can see this quotation
Internal sales manger can make changes on quotation and confirm it
Portal sales user can see that his quotation is confirmed and converted to sale order
Portal sales user can log notes in the quotation or sales order
